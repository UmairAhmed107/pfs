Two-year campus placement plan

One foundation.
Three ways to stand out.

Python full stack and DSA stay constant for the whole two years. The differentiator skill is the one fork in the road — pick where you want to specialize once the foundation is solid.

Shared · Months 1–4

Foundation

Programming & web basics: Get comfortable with Python, then HTML, CSS, and core JavaScript before touching any framework. Learn Git and GitHub properly — every project from here on lives there.

DSA: Time/space complexity intuition, arrays, strings, basic searching and sorting, simple recursion. Correct mental models first, speed comes later.

Web dev: Start React — components, props, state, hooks. In parallel, start Django (models, views, ORM) or FastAPI. Build two or three throwaway projects just to get the front-end/back-end connection comfortable.

Aptitude: Light, regular practice — 20–30 minutes a few times a week. It compounds over two years with barely any dedicated focus later.

Choose a track to expand

Across all three tracks

One isn't exclusive. Most strong candidates pick up light exposure to a second area along the way — someone deep in AI integration often learns basic cloud deployment just to ship their project.

Internships outrank any single skill. If a real internship comes up in phase 3 or 4, it's usually worth prioritizing over additional self-study time.

DSA stays constant. Practice a little most days for the full two years rather than cramming — it's tested almost identically regardless of which track you choose.
